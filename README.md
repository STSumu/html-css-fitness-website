# Fitness Landing Page

A responsive fitness landing page built using **HTML** and **CSS** as part of a web development course assignment. This project was created to practice modern layout techniques, responsiveness, and UI structuring.

## Features

* Responsive navigation bar with mobile menu
* Hero section with call-to-action buttons and info cards
* “Change Your Habits” section using grid layout
* Training and exercises gallery
* BMI calculator UI section
* Animated diet promotion section
* Team showcase section with background images
* Fully responsive design for mobile and tablet devices

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* CSS Animations
* Google Fonts

## Project Highlights

* Built a complete multi-section landing page
* Used Flexbox and Grid together for complex layouts
* Implemented responsive design for multiple screen sizes
* Added CSS animations for interactive UI effects
* Followed a real-world UI design structure

## Learning Outcome

Through this project, I improved my understanding of:

* Layout structuring in large web pages
* Responsive design techniques
* CSS positioning and animations
* UI consistency and spacing
* Converting a full UI design into code
